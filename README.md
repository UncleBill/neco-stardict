about
-----
A [neocomplcache.vim][] plugin for dictionary looking up on the fly

install
-------
recommend [vundle][] or [neobundle][]
add `Bundle "UncleBill/neco-stardict"` to your `.vimrc`

requirements
------------
- [stardict][] installed
- [sdcv][] installed
- [懒虫简明英汉词典][dictionary](click to download) installed

inspired
--------
